Responsive facebook gallery, just plug and play

== Description ==

Responsive Facebook Gallery helps you to integrate facebook gallery easily to your website in few easy steps.

It is compatible with all major browsers IE8+, firefox, chrome, safari, opera and android browsers

Once its activated enter the shortcode to the page where you want to display gallery.

An online demo can be found at http://pixsols.com/test/wordpress/responsive-facebook-gallery/.

Need help or want to customize this plugin? Write us at http://www.pixsols.com/contact-us/

== Installation ==

1. Unzip and upload the `facebookgallery` folder to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Create Facebook APP and get APP ID and Secret Key - Documentation: https://smashballoon.com/custom-facebook-feed/access-token/
4. Genarate your Access Tocken  - https://smashballoon.com/custom-facebook-feed/access-token/ (A bottom of the page)
5. Now go to Settings -> Responsive Facebook Gallery and enter APP ID, Secret Key and Access Tocken.
6. Use the shortcode `[fbgallery url='FACEBOOK GALLERY COMPLETE URL' column='{1,2,3,4}']` in your posts or pages.
7. You are done.
